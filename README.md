# Security Jobs Radar

Enter a company domain to discover what their security hiring activity reveals about current priorities, tooling stack, compliance focus, and team maturity.

## Features

- **Automatic careers page discovery** — tries common paths and detects ATS platforms (Greenhouse, Lever, Ashby, Workable, etc.)
- **Security role filtering** — identifies security-related positions from all job listings
- **Multi-dimensional analysis:**
  - Security domains (AppSec, Cloud Security, GRC, SOC/IR, DevSecOps, etc.)
  - Tools & platforms (Splunk, CrowdStrike, Snyk, Okta, etc.)
  - Compliance frameworks (SOC 2, ISO 27001, PCI DSS, HIPAA, GDPR, etc.)
  - Urgency and pressure signals
  - Team maturity indicators
  - Seniority distribution
- **Intel Brief** — auto-generated summary of key findings
- **Export** — copy report or download as JSON
